## How to Contribute

### Step 1: Fork the Repository
1. Click the "Fork" button at the top right of this repository
2. This creates a copy in your GitHub account

### Step 2: Clone Your Fork
```bash
git clone https://github.com/YOUR-USERNAME/plushub.git
cd plushub
```

### Step 3: Create a Branch
```bash
git checkout -b update-streaming-page
# Or: git checkout -b add-new-resource
```

### Step 4: Make Your Changes
- Edit the markdown files you want to update
- Add new resources to appropriate categories
- Update existing information

### Step 5: Commit Your Changes
```bash
git add docs/streaming.md  # or whatever file you edited
git commit -m "Update: Add new streaming sites"
```

### Step 6: Push to Your Fork
```bash
git push origin update-streaming-page
```

### Step 7: Open a Pull Request
1. Go to your fork on GitHub
2. Click "New Pull Request"
3. Select your branch
4. Fill out the PR description
5. Submit the PR

### Step 8: Wait for Review
- We'll review your changes
- We may request modifications
- Once approved, your changes will be merged! ğŸ‰
